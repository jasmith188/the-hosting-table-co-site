# The Hosting Table Co. React + Vite Site

A polished private dining landing page for The Hosting Table Co. using React, Vite, Outfit from Google Fonts, and Netlify-ready configuration.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Netlify

1. Push this folder to GitHub.
2. In Netlify, choose **Add new site > Import an existing project**.
3. Connect the GitHub repo.
4. Build command: `npm run build`
5. Publish directory: `dist`

The included `netlify.toml` already sets this up.
